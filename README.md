# Memory Matching Game

A simple interactive Memory Matching Game built with **HTML, CSS, and jQuery**.

## 🎮 Features

- Flip cards to find matching pairs
- Only 2 cards can be flipped at a time
- Track number of moves
- Timer shows time taken to complete the game
- Sound effects for matches and wrong moves
- Win message when all pairs are matched
- Smooth flip animation

## 🛠️ Skills Practiced

- jQuery selectors and events
- DOM manipulation
- Game state management
- CSS animations & effects
- HTML5 Audio API
- Problem-solving with interactive applications

## 📝 How to Play

1. Open `index.html` in your browser.
2. Click on two cards to reveal symbols.
3. If the symbols match, they stay face-up, and a match sound plays.
4. If they don’t match, they flip back after 1 second with a wrong sound.
5. Try to match all pairs in the fewest moves and fastest time!
